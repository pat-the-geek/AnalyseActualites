# Documentation du dossier `samples/`

Le dossier `samples/` contient des exemples de rapports générés par le pipeline AnalyseActualités (par exemple : `articles_generated_2026-02-01_2026-02-28.json`).

Ces fichiers servent de référence pour la structure de sortie attendue et sont utilisés pour illustrer le fonctionnement du projet. Certains de ces exemples sont également publiés sur GitHub pour faciliter la démonstration et la validation du pipeline.
